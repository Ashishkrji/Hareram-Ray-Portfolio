---

## 🚀 Overview & Features

Inspired by top-tier modern layout aesthetics, this portfolio showcases academic milestones, specialized certifications, internship experiences, and core competencies in talent management, recruitment strategies, brand promotion, and business analytics.

### ✨ Key Highlights
- **Stunning Rich Aesthetics**: Built with vibrant accent colors (`#0f766e`), sleek typography (`Inter`), and modern glassmorphism UI components.
- **Dynamic Micro-Interactions**: Fully responsive navigation, smooth scrolling, scroll-triggered fade animations, and interactive stat counters.
- **Floating Contact Panel**: Quick-access floating action buttons for instant WhatsApp messaging and smooth Back-to-Top scrolling.
- **High-Performance Architecture**: Single-file structure utilizing Alpine.js for lightweight reactivity and Tailwind CSS via CDN for rapid styling without heavy build dependencies.

---

## 🛠️ Technology Stack

- **HTML5**: Semantic, accessible markup optimized for SEO.
- **Tailwind CSS**: Modern utility-first styling framework configured with a bespoke teal color palette.
- **Alpine.js**: Lightweight JavaScript framework for responsive drawers, scroll tracking, and modal state management.
- **Google Fonts**: Crisp and highly readable `Inter` font family.

## 💻 Quick Start & Deployment

This project requires zero build configuration. It can be viewed instantly or deployed to any static web hosting provider.

### Local Execution
1. Clone the repository:
   ```bash
   git clone https://github.com/Ashishkrji/Hareram-Ray-Portfolio.git
   ```
2. Open `index.html` directly in any modern web browser (e.g., Google Chrome, Microsoft Edge, Firefox, Safari).

### Deployment Options
Easily host for free on top cloud platforms:
- **Vercel / Netlify / Cloudflare Pages**: Simply link your Git repository.
- **GitHub Pages**: Go to Repository Settings > Pages > Select `main` branch.
